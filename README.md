README
------

This project provides a lovely kivy interface for ATM4COIN

Dependencies
------------

Kivy >= 1.8.0
zbarcam (qr scanner)
devilspie2

# Install zbar
    on mac:
    $ brew install zbar

    on ubuntu:
    $ apt-get install zbar-tools


Setup Instructions
------------------

*   RaspberryPi (2 B+ recommended)
*   Olimex (A20 Micro recommended)
